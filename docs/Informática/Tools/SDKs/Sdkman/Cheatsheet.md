# Cheatsheet SDKMAN

## Instalar una JDK externa para que se pueda gestionar con SDKMAN

* Nos bajamos la Sdk donde queramos; ejemplo: `~/.bin/Sdks/Java/jdk1.8.0_202/`
* La instalamos con: `sdk install java ${nombre} ${path}`
* Ejemplo: `sdk install java oracle-1.8.0.22 ~/.bin/Sdks/Java/jdk1.8.0_202/`
