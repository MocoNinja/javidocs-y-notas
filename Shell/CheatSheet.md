# Cosas rápidas de la shell de UNIX

## Comandos rápidos y derivados

* Código de salida del comando anterior: ```$?```

## Cosicas Específicas

### IDS de Usuario y Grupo

* UID: ```id -u username```
* GID: ```id -g username```
* Todos los grupos de un usuario: ```id -G username```
* I want it all: ```id username```
